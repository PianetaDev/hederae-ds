## Configurazione SSH (da ripristinare in ogni sessione)

```bash
# 1. Ricrea i file SSH (incolla la chiave privata quando richiesto)
mkdir -p ~/.ssh && chmod 700 ~/.ssh

# Incolla la chiave privata ed25519 nel file:
# cat > ~/.ssh/hederae_deploy << 'PRIVKEY'
# -----BEGIN OPENSSH PRIVATE KEY-----
# [INCOLLA QUI LA CHIAVE PRIVATA]
# -----END OPENSSH PRIVATE KEY-----
# PRIVKEY
chmod 600 ~/.ssh/hederae_deploy

# 2. Configura SSH
cat > ~/.ssh/config << 'SSHCFG'
Host github-hederae
  HostName ssh.github.com
  Port 443
  User git
  IdentityFile ~/.ssh/hederae_deploy
  IdentitiesOnly yes
SSHCFG
chmod 600 ~/.ssh/config

# 3. Imposta il remote HTTPS (usa il PAT salvato in password manager)
cd /sessions/peaceful-relaxed-faraday/mnt/26P12_Hedera/hederae-ds
git remote set-url origin https://[PAT]@github.com/PianetaDev/hederae-ds.git
git config http.proxy http://127.0.0.1:3128

# 4. Pusha
git push origin main
```

## Note

- SSH porta 443 usata perché porta 22 è bloccata nel sandbox Cowork
- Proxy HTTP su 127.0.0.1:3128 — git configurato di conseguenza
- Se il push HTTPS fallisce, usare il metodo SSH sopra
